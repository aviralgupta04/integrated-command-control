# Integrated Command and Control (ICC)

## 📖 Overview

Integrated Command and Control (ICC) is a centralized surveillance and monitoring platform designed to manage cameras, live streams, recordings, alarms, and users from a single dashboard.

The project is being built using **React** for the frontend and **ASP.NET Core (.NET)** for the backend while following clean architecture principles.

---

## 🚀 Tech Stack

- Frontend: React
- Backend: ASP.NET Core (.NET)
- Language: C#
- Database: SQL Server
- Authentication: JWT
- Realtime Communication: SignalR (Planned)

---

## 📌 Planned Features

- User Authentication
- Role-Based Access Control
- Dashboard
- Camera Management
- Live Video Streaming
- PTZ Camera Control
- Recording Management
- Alarm Monitoring
- Device Health Monitoring
- Audit Logs
